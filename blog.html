What is the Document Object Model (DOM)?

The DOM is a programming interface that represents an HTML or XML document as a tree structure, allowing scripts to dynamically access, manipulate, and modify the document's content, structure, and styles.


How do you select an element from the DOM?

You can select an element using methods like: document.getElementById('id') document.querySelector('selector') document.querySelectorAll('selector') document.getElementsByClassName('className') document.getElementsByTagName('tagName')


What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events for its child elements. It improves performance and handles dynamically added elements effectively.


How do you manipulate an element's attributes and styles using the DOM?

Use element.setAttribute('attribute', 'value') to set attributes. Use element.style.property or element.classList to modify styles .
